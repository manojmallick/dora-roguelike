# DORA: The Compliance Roguelike

Survive the audit. Save the FinTech.

## Local Development

```bash
npm install
npm run dev
```

Open the Vite URL and tap once to unlock audio. Until generated ElevenLabs files are present, keep `VITE_DEBUG_SKIP_AUDIO=true`.

## QA Shortcuts

- `N`: next round
- `B`: force boss round
- `V`: force victory
- `F`: force defeat
- `Q/W/E/R/T`: damage compliance indicators
- `A`: restore the lowest indicator
